Spring Cloud Eureka Client 向注册中心注册

2017-12-17

Spring Boot: 1.5.10.RELEASE

Spring Cloud: Dalston.SR5

依赖的 jar

application.yml 添加配置

添加如下配置：

eureka.client.service-url.defaultZone : 需要填写 Eureka 注册中心真实的地址，可以是 IP 加端口，也可以使用域名

application 启动类

需要添加 @EnableEurekaClient 注解，表示本应用是 Eureka 客户端

访问注册中心

访问地址：http://localhost:8761

可以发现 Instances currently registered with Eureka 的内容发生变化，出现一条记录，其中：

Application 的名称即是 Client 应用的 spring.application.name 属性值，Eureka 会自动将其装换为全部大写

代码可参考 lxmuse-spring-cloud 下 example-spring-cloud 的 example-eureka-02 模块
