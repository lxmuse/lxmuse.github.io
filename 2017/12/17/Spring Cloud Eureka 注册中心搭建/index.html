Spring Cloud Eureka 注册中心搭建

2017-12-17

Spring Boot: 1.5.10.RELEASE

Spring Cloud: Dalston.SR5

依赖的 jar

application.yml

spring.application.name : 应用名称

server.port : 应用端口

eureka.instance.hostname : Eureka 实例名称

eureka.client.register-with-eureka : 表示不向注册中心注册自己

eureka.client.fetch-registry : 表示不需要检索服务列表

eureka.client.service-url.defaultZone : 表示 Eureka 注册中心地址

application 启动类

需要添加 @EnableEurekaServer 注解，表示本应用是 Eureka 服务端

访问注册中心

访问地址：http://localhost:8761

主要是关注 Instances currently registered with Eureka 的内容
当前 No instances available 表示还没有服务注册到注册中心

代码可参考 lxmuse-spring-cloud 下 example-spring-cloud 的 example-eureka-01 模块
