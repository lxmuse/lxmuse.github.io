Java ServletConfig 和 ServletContext

Servlet : 3.1

ServletConfig：

ServletConfig 持有了 Servlet 的初始化参数，Servelt init 方法被调用时需要 ServletConfig，也可以通过 getServletConfig 直接获取 ServletConfig。

ServeltContext：

ServletContext 是应用上下文参数，用于应用内不同的 Servlet 用于共享资源信息。

一个应用可以拥有多个 Servlet，每个 Servlet 都拥有自己的 ServeltConfig，但是只能拥有一个 ServletContext
