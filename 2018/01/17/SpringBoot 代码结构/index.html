SpringBoot 代码结构

SpringBoot：1.5.10.RELEASE

不要使用默认包

default package 不能使用，SpringBoot 特性是自动扫描，自动装配，对代码结构有一定的约定，default package 结构可能影响自动扫描，自动装配

启动类的位置

官方推荐启动类位于根目录，因为 @EnableAutoConfiguration 注解是基于当前类所在包，扫描当前类所在包及子包
